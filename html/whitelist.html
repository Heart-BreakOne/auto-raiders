Set you priority captains lists

  • When switching an idle captain the order of priority is: Masterlist (if enabled to do so) > Diamond > Gold > Silver > Bronze > Allowlist > Favorites > Any captain
  • Blocklisted captains will be ignored, unless they are the only ones in the Any Captains list.
  • Write the name of the Captains with a space in between
  • The order you write the captains is the order of their priority
  • When you blocklist a captain, make sure they are not in the in-game favorites list
  • Uppercasing and lowercasing don't matter, but the captain name MUST be typed correctly (e.g CaptainZ1, captainz1, CAPTAINZ1 are all valid because their only difference is the casing)
  • Masterlist has two states and bypass priority, read instructions














Dungeons

The playstyle and rigidness of some captains change when they are doing Dungeons
When slot 0 gets set to Dungeons, these lists will govern it.
Priority logic remains the same: Dungeon list > Diamond > Gold > Silver > Bronze > Favorites > Any
Ignore list captains will be completely ignored even if there are no other captains available.
