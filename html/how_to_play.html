SRHelper is a click emulator for the game Stream Raiders.

Download for Google Chrome at the extension store here


To get started, load the game after installing the extension and click on the extension icon on the extensions tab. That will open the SRHelper popup screen.
The popup has several settings to tailor the gameplay to your needs. Some of them are mandatory such as which units are allowed to be placed.
Below is the explanation of all the extension settings and how they work:

Reloader Due to memory and caching, the game eventually becomes unstable and unresponsive until the tab unrecoverably crashes. To mitigate this effect, set a period in minutes for the website's page to automatically refresh so the game is cleared from memory. Default is 60 minutes.
UPDATE MAP!! If are you playing with loyalty preservation ON you will see the warning "UPDATE MAP!!" on the top header of the game whenever there's a game update. Go to the extension icon and click Update Maps, follow the instructions there to update the maps. If you are not preserving loyalty this warning can be ignored, after updating the map the warning goes away after the first loyalty battle.
Switch idle captain: It will always switch to a campaign captain Switch idle captain after 15 minutes, the idle states are: if a captain doesn't find a battle, doesn't start a battle or doesn't collect the reward.
Leave slot empty if there aren't any loyalty captains to switch After removing an idle captain, by default the extension will seek a captain with loyalty: diamond, gold, silver, then bronze. If it doesn't find any it will seek a favorited captain. If doesn't find any it will randomly select one, if this option is enabled it won't select a random captain if a loyalty/favorite captain isn't available.
Buy Scrolls Automatically buy scrolls from the store.
Buy extra $100 scrolls Refresh the store for 100 coins and buy the scrolls.
Collect quests Collect completed quests.
Collect daily reward Collect free daily event rewards such as cupcakes.
Equip captain skins Automatically equip captain skins
Collect battlepass Collect completed battlepasses (it will not collect potions if the amount of potions is maxed out).
Placeable units
Use common units Allow the placement of common units.
Use uncommon units Allow the placement of uncommon units.
Use rare units Allow the placement of rare units.
Use legendary units Allow the placement of legendary units.
Throttle to place only one unit.
Unlimited campaign Place more than one unit on campaign mode.
Unlimited duels Place more than one unit on duel mode.
Unlimited clashes Place more than one unit on clash mode.
Unlimited dungeons Place more than one unit on dungeon mode.
Loyalty preservation.
Don't preserve loyalty Switch or place with a captain regardless of loyalty.
Preserve bronze loyalty Switch or place with a captain if they are at least bronze.
Preserve silver loyalty Switch or place with a captain if they are at least silver.
Preserve gold loyalty Switch or place with a captain if they are at least gold.
Preserve diamond loyalty Switch or place with a captain if they are at least diamond.
Live captains and Masterlist
Force live switch to masterlist If a slot has a captain that isn't masterlisted, it will be switched on the first opportunity.
Force live switch to priority masterlist Switch a masterlisted captain to another masterlisted captain that is in front of the list.
Idle captains and Masterlist
Prioritize masterlist on idle switch Masterlist will have priority over diamond, gold, silver, bronze, allowlist and favorite list.
Skip idle switch if no masterlist is online If there are no masterlist captains online, leave the slot empty.
Epic units/Potion usage
Potion allowlist Use potions only if the captain is potionlisted. Potion usage must be set to 45 or 100.
No potions No potions will be used.
Potions at 45 If there's a minimum of 45 potions, epic units will used.
Potions at 100 If there's exactly 100 potions, epic units will be used.
Color code: From time to time you'll see the slots changing colors, here's what they mean. Units won't be placed on colored slots, new placement attempts are made every 3 minutes
Red It means that a slot is running a special mode (dungeons/PVP) while a second captain switched to the same mode, the second captain will be colored red and no units will be placed to avoid resetting keys for example. It also means that a captain was running a special mode and switched to campaign, some users dislike this because it locks the slot for 30 minutes, no units will be placed to preserve the slot.
Purple There aren't any units available to be placed on the available markers, this can be due to units on cooldown or unmatching markers.
Blue If the preserve diamond loyalty option is enabled and the captain is on a loyalty chest and doesn't have loyalty no units will be placed.
Wipe data button Reset the red, purple and gold states as well as the enabled/disabled buttons.
ENABLED/DISABLED idle switcher buttons Pause slot placement and offline switching.
